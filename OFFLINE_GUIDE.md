# 离线使用与安装指南

本项目现已支持完全离线使用，并可作为桌面应用“安装”到电脑上。

## 📥 方式一：直接运行 (无需安装)
1.  确保 `libs` 文件夹与 `index.html` 在同一目录下。
2.  双击 `index.html` 即可在浏览器中打开。
3.  **断网测试**：即使断开网络，所有功能（生成、导入Excel、导出ZIP等）均可正常使用。

## 🖥️ 方式二：安装为桌面应用 (PWA)
使用 Chrome 或 Edge 浏览器打开页面后，你可以将其安装到桌面，获得类似原生 App 的体验：

1.  打开 `index.html` `(注意：PWA通常需要通过本地服务器或HTTPS访问才能触发安装，直接双击file://可能不会显示)`。
2.  查看浏览器界面：
    *   **地址栏右侧**：可能会出现一个“安装”图标（通常是带加号的小电脑）。
    *   **页面右上角**：部分浏览器（如最新版 Edge）可能会在工具栏右上角显示“应用可用”图标。
    *   **菜单中**：点击浏览器右上角的三个点 (⋮)，菜单中也可能有“安装 送货单QR生成器”的选项。
3.  点击并选择 **"安装"**。
4.  现在，你可以从桌面图标直接启动它，它将在独立的无边框窗口中运行，非常清爽！

## 📂 部署到 GitHub
上传时，请确保上传以下所有文件和目录：
*   `libs/` (文件夹及内部所有文件)
*   `index.html`
*   `styles.css`
*   `script.js`
*   `enhancements.js`
*   `enhancements.css`
*   `qrcode.min.js`
*   `manifest.json`
*   `sw.js`

*(注意：不再需要上传 `script-integration-guide.md` 和 `enhancements-functions.js`)*
