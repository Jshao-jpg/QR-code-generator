# 高级送货单二维码生成器 (Advanced QR Code Generator)

专为送货单管理设计的批量二维码生成工具，支持双模式生成、智能校验与精细化 Excel 导出格式。

## ✨ 核心功能特性

### 1. 双模式一体化界面
*   **送货单表头 (DN Header)**: 包含 DN No., Vendor ID, PO No.，支持单独管理。
*   **送货单明细 (DN Detail)**: 包含 Full PO No., Qty, Unit, Unique ID, PN。
*   **统一操作**: 只需点击一次生成，即可同时渲染表头和明细的所有二维码！

### 2. 智能数据输入 (Smart Paste)
*   **启发式粘贴解析**: 支持直接从 Excel 复制表头或明细数据并在同一框内粘贴。系统不仅能自动识别 Tab 或空格分隔符，还可根据 `DN No`, `送货单号` 等关键字自动判断数据归属（表头还是明细）。
*   **模板导入与手动录入**: 提供标准 Excel 模板下载与导入，并保留高交互性的手动录入模式（获焦呈现专属绿色视觉）。

### 3. 严格且灵活的数据校验 🛡️
*   **宽容查重机制**: 实时检测重复数据。但只要**流水号 (Unique ID)** 不同，即允许并存，满足多箱同料的打印需求。
*   **格式自适应校验**: 强制检查 `Full PO No.` 格式（必须包含 `-`），智能检查数量正数。

### 4. A4 纸级优化的精确报表导出 🚀
*   **打印级缩放 (`enhancements.js`)**: 生成的 Excel 模板在栏宽（采购单号、料号等）和行高上历经精细缩小，100% 适配单页 A4 纵向打印。
*   **绝对居中锚定技术 (EMUs)**: 底层由 ExcelJS 驱动，通过计算原生的 EMUs (`nativeColOff`, `nativeRowOff`)，严格实现了 70x70 像素清晰二维码在狭小单元格内不受拉伸地完美绝对剧中。

### 5. 跨端 PWA 应用级支持 📱
*   **桌面安装**: 利用 Service Worker 和 `beforeinstallprompt` 机制。使用现代浏览器打开时，可通过“安装到桌面”按钮将此工具安装为本地应用程序。
*   **极速响应与全脱机**: 一旦打开，网络波动不影响缓存核心数据的使用与生成。

## 🛠️ 使用指南

### 快速开始
1.  **输入数据**:
    *   *方法 A (模板)*: 点击蓝色“下载模板”，在本地 Excel 填好后“导入数据”。
    *   *方法 B (智能)*: 在任意表格中选中明细，在紫色的“快速粘贴”框内 `Ctrl+V` ，点击解析并填充。
2.  **生成**: 点击底部引人注目的紫色“批量生成所有二维码”按钮。
3.  **结果导出**:
    *   📥 **Excel 明细导出**: 点击“导出送货单明细表”，获取内置精确排版和精美二维码的专属 `xlsx` 打印表。
    *   📦 **ZIP 图像包**: 一键打包下载全部图码原文件。

### 快捷键与技巧
*   **点击二维码**: 随时点击界面上的二维码查看高清大图。
*   **状态保存屏蔽**: 刻意不留历史脏数据，每次刷新保障表格清空，开箱即用。

## 📂 文件结构

*   `index.html` - 主界面（暗色玻璃磨砂 UI 设计）
*   `styles.css` - 核心样式（蓝/紫/绿三色专属功能引导）
*   `script.js` - 主逻辑与交互控制、智能粘贴识别器
*   `enhancements.js` - Excel 模板精准对位渲染器
*   `sw.js` & `manifest.json` - PWA 脱机及缓存刷新机制
*   `libs/` - 本地化的前沿依赖库 (qrcode, exceljs, jszip, FileSaver)

## 📦 部署说明

本项目为纯静态构建，无需 Node.js 后台。
*   **本地使用**: 推荐搭建简单的静态 Server（例如 VSCode Live Server）以完全激活 PWA 支持。
*   **内网发布**: 直接一键打包拖入到任意 Nginx, Tomcat 或 GitHub Pages 服务目录即可。
