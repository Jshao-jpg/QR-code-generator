# 高级送货单二维码生成器 (Advanced QR Code Generator)

专为送货单管理设计的批量二维码生成工具，支持双模式生成、智能校验与多种导出格式。

## ✨ 核心功能特性

### 1. 双模式支持
*   **送货单表头 (DN Header)**: 包含 DN No., Vendor ID, PO No.
*   **送货单明细 (DN Detail)**: 包含 Full PO No., Qty, Unit, Unique ID, PN

### 2. 智能数据输入
*   **Excel 粘贴**: 支持直接从 Excel 复制数据粘贴，自动识别 Tab 或空格分隔符。
*   **模板导入**: 提供标准 Excel 模板下载，支持文件拖拽或点击上传。
*   **手动输入**: 支持直接在网页表格中编辑数据。

### 3. 严格的数据校验 🛡️
*   **格式校验**: 强制检查 `Full PO No.` 格式（必须且只能包含两个连字符 `-`，且中间需有内容）。
*   **查重机制**: 实时检测重复数据，并精确提示“第 X 行与第 Y 行重复”。
*   **数字校验**: 自动检查数量 (Qty) 是否为有效正整数。

### 4. 强大的导出与交互 🚀
*   **批量导出**:
    *   📥 **ZIP 打包**: 一键打包下载所有二维码图片。
    *   📋 **网格复制**: 将多个二维码拼合成一张网格图复制到剪贴板（方便粘贴到邮件/文档）。
    *   🖼️ **SVG 矢量图**: 导出矢量格式，无限放大不失真。
    *   💾 **PNG 下载**: 批量下载标准图片。
*   **预览功能**: 点击二维码可查看高清大图（自适应紧凑布局）。
*   **打印优化**: 支持 `Ctrl + P` 打印模式，自动隐藏无关 UI，只按清单打印二维码。

## 🛠️ 使用指南

### 快速开始
1.  **选择模式**: 页面上方选择“送货单表头”或“送货单明细”。
2.  **输入数据**:
    *   *方法 A*: 点击“下载模板”，填好后点击“导入数据”。
    *   *方法 B*: 在 Excel 中选中数据复制，点击页面上的“Excel 粘贴解析”。
3.  **生成**: 点击“批量生成”按钮。
    *   *系统会自动进行校验，如有错误或重复会弹出提示。*
4.  **导出**: 根据需要选择“下载全部”、“复制全部”或“打包下载”。

### 快捷键与技巧
*   **Ctrl + P**: 进入打印预览模式。
*   **点击二维码**: 查看大图（支持直接复制/下载单张）。

## 📂 文件结构

*   `index.html` - 主界面结构
*   `styles.css` - 核心样式与打印布局
*   `script.js` - 主逻辑与交互控制
*   `enhancements.js` - 增强功能模块（校验、打包、预览等）
*   `libs/` - 依赖库 (qrcode, xlsx, jszip, FileSaver)

## 📦 部署说明

本项目为纯静态 Web 应用，无需后台服务器。
*   **本地使用**: 直接双击打开 `index.html`。
*   **网络部署**: 可直接部署至 GitHub Pages, Vercel, 或任意静态文件服务器。

## 💻 浏览器兼容性

推荐使用 **Chrome**, **Edge**, 或 Firefox 以获得最佳体验（Clipboard API 和高级打印支持）。
